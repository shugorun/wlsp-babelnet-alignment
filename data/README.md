# Data Policy

このリポジトリでは、データと生成物をそのまま全面公開するのではなく、ライセンスと再配布条件に合わせて制限しています。

## 含めないもの

- BabelNet データ本体
- BabelNet 由来の processed / interim data
- API 入出力の生データ
- 公開に適さない raw data

## 理由

- WLSP は CC BY-NC-SA 3.0 の条件に従う必要があります。
- BabelNet は独自ライセンスがあり、公開 GitHub にそのまま再配布すべきではないデータや派生物を含みます。

## 公開版に残すもの

- コード
- 実験要約
- 軽量な設定ファイル
- ドキュメント

## 利用する場合

各自でライセンスを確認したうえで、必要なデータを取得し、ローカル環境で配置してください。
